Learn more at: [https://adventofcode.com](https://adventofcode.com)

### How to Use
1. Navigate to the folder for the specific year and day.
2. Open `problem.md` to read the puzzle description.
3. Check `input.txt` for the puzzle input.
4. Run `main.py` to execute the solution.

### Example Workflow
```bash
# Clone the repository
git clone https://github.com/icequezon/advent-of-code.git
cd advent-of-code/2025/01

# Run the solution for Day 1 of 2025
python main.py
```
